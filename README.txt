코드를 실행하기 위해서는
Node.js + express + p5.js + socket.io + d3.js 를 사용할 환경을 갖추어야 합니다.
그리고 final_code.zip의 압축을 푼 후
cmd에서 server.js가 있는 폴더로 이동 후 "node server.js" 명령어로 서버를 오픈할 수 있습니다.
그리고 sketch.js의 ip주소를 해당 PC의 IP주소로 변경하여야 합니다 // socket = io.connect('http://192.168.0.7:3000'); 
그리고 result_page.html의 ip주소 역시 해당 PC의 IP주소로 변경하여야 합니다 // <a href='http://192.168.0.7:3000/' target="_top">
